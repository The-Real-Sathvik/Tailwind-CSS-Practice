Create a Donut Shape Using Tailwind CSS

Instructions:
-> Create a div element to represent the donut shape.
-> Use Tailwind CSS utilities to:
-> Make the div circular by applying equal width and height with w-40 h-40 or other sizes.
-> Use the rounded-full class to give the element a circular shape.
-> Apply a background color to the outer donut using classes like bg-red-500.
-> Make the div circular by applying equal width and height with w-16 h-16 or other sizes.
